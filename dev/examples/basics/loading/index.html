Loading Images

We'll start by downloading a sample image. If you have an image stored locally, you would skip this step.

using AstroImages
fname = download(
    "http://www.astro.uvic.ca/~wthompson/astroimages/fits/656nmos.fits",
    "eagle-656nmos.fits"
);

Load the image by filename. If unspecified, the image is loaded from the first image-HDU in the fits file.

img = AstroImage("eagle-656nmos.fits")
